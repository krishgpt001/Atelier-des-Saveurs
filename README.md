# Atelier des Saveurs — Restaurant Landing Page
A fully responsive single-page restaurant website built with HTML, CSS, JavaScript and Bootstrap 5.

## Project Structure
```
Atelier des Saveurs/
├── index.html
├── css/
│   ├── bootstrap.min.css
│   ├── custom.css
│   └── responsive.css
├── js/
│   ├── bootstrap.bundle.min.js
│   └── main.js
└── assets/
    ├── images/
    └── fonts/
```

## How to Run Locally
No build tools required. Open `index.html` directly or run a simple local server.

Install (VS Code Live Server):
```bash
# Install Live Server extension → Open `index.html` with Live Server
```

## Features Implemented
- Sticky navbar with scroll state
- Smooth scroll and active section highlighting
- Gallery lightbox (Bootstrap modal)
- Testimonials carousel (Bootstrap)
- Menu filter tabs (JS)
- Contact / reservation form with client-side validation and success state
- Today's Special badge (shows on configured days)
- Accessibility basics: semantic tags, alt text, ARIA attributes
- Basic SEO meta tags and Open Graph tags in `index.html`
